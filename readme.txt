A fully customizable Javascript countdown

== Description ==

Javascript Countdown allows to insert a countdown anywhere on your Wordpress website.

The Javascript Countdown is fully customizable from your Wordpress admin.  There, you can personalize the date, time, the format of the countown and the final message.  You can even include any HTML you want !



== Installation ==

1. Upload the Javascript Countdown
2. Activate the plugin
3. Modify the plugin in the admin (using the Javascript Countdown menu link)
4. Insert the function <? show_countdown (); ?> anywhere in your website (header, post, template)
